**Ghostty + Claude Code with SAND Keybindings**

Reference: https://x.com/i/status/2023151176758268349

SAND mnemonic for panel management:

- **S**plit panels
  - `Cmd+D` — split right (vertical)
  - `Cmd+Shift+D` — split down (horizontal)

- **A**cross tabs
  - `Cmd+T` — new tab
  - `Cmd+Shift+←/→` — navigate between tabs

- **N**avigate splits
  - `Cmd+Alt+Arrows` — jump in any direction
  - `Cmd+Shift+E` — equalize all splits
  - `Cmd+Shift+F` — zoom/restore current panel

- **D**elete panels
  - `Cmd+W` — close current panel or tab

**Setup (deferred - not implemented today):**
1. Download Ghostty from ghostty.org (Linux AppImage or build from source)
2. Create config dir: `mkdir -p ~/.config/ghostty`
3. Download config: `curl -o ~/.config/ghostty/config https://gist.githubusercontent.com/davila7/5b07f55a6e65a06c121da9702d10c2e2/raw/config`

**Usage pattern:**
- Left panel: Claude Code
- Right panel: Lazygit (monitor commits/diffs in real-time)
- Right-bottom panel: File browser (e.g., yazi)
- Can scale to multiple Claude Code instances using git worktrees
